# 美业店务 SaaS 系统

覆盖轻医美、生美、日化零售的智能店务与营销管理系统。

适用于：轻医美、生美、美容、美发、美甲、按摩、瑜伽、培训、宠物等行业。  
支持：单店、连锁店、加盟店。

## 产品亮点

- 多业态支持：轻医美、生美、日化零售等
- 多端协同：后台管理端、门店 APP、小程序
- 店务一体化：预约、会员、收银、库存、报表、营销
- 连锁化管理：门店、员工、权限、数据权限统一配置
- 营销与客户运营：会员分群、卡券、促销、回访、客户雷达

## 技术栈

| 层次 | 技术/组件 | 用途 |
|---|---|---|
| 编程语言 | Java | 服务端业务逻辑实现 |
| 基础框架 | Spring Boot | 应用快速开发与自动配置 |
| 微服务框架 | Spring Cloud Alibaba | 服务注册、配置、熔断、限流 |
| 服务网关 | Spring Cloud Gateway | 统一接入、路由、鉴权前置 |
| 接口文档 | Springdoc + Knife4j | OpenAPI 3 接口文档与调试 |
| ORM 框架 | MyBatis Plus | 数据持久化与分页 |
| 连接池 | Druid | 数据库连接监控 |
| 缓存 | Redis / Redisson | 分布式缓存与锁 |
| 消息队列 | RocketMQ | 异步事件与削峰 |
| 定时任务 | xxl-job | 分布式任务调度 |
| 权限安全 | Spring Security + OAuth2 / JWT | 统一身份认证与权限控制 |
